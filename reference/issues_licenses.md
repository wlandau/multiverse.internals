# Report package license issues

Report packages without standard free and open-source licenses.

## Usage

``` r
issues_licenses(meta = meta_packages())
```

## Arguments

- meta:

  Package metadata from
  [`meta_packages()`](https://r-multiverse.org/multiverse.internals/reference/meta_packages.md).
