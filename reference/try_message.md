# Get try error message

Get the error message of an error object from
[`try()`](https://rdrr.io/r/base/try.html).

## Usage

``` r
try_message(try_error, collapse = " ")
```

## Arguments

- try_error:

  [`try()`](https://rdrr.io/r/base/try.html) error object.

- collapse:

  Character of length 1, delimiter for the lines of the message.
